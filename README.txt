------- Twitter Automation ---------------

Installing the project:

1. Open your preferred Python IDE

2. Open the project

3. Import dependencies (selenium)
	pip install selenium (if not installed already)

4. Install webdriver manager
	open terminal
	pip install webdriver-manager

5. On the TwitterAutomation project, open folder "src" and within directory "tests" you may choose "TestTwitter.py"

6. Update the path for media files
	On lineS 94 and 131 please update the path accordingly to 	where your project is located. You may edit the section of the path "C:/Users/de/PycharmProjects" and replace it so it can be like this "<your-path>/TwitterAutomation/src/assets/image.png"

7. After updating the path you may proceed to run the project.

------- Additional Features ---------------

Changing login information: If you would like to, you may update the credentials by changing the values in the "login.enter_username" elements.

Changing Tweet Input: If you would like to update the Tweet Contents you may do so by changing the content on the "home.input_tweet" elements that are available in the TestTwitter file.
